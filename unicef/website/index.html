About This Application

Weather Forecast System provides accurate 15-day temperature predictions for Maharashtra districts using artificial intelligence.

Features:

  • 15-day temperature forecasts
  • District-specific predictions
  • Real-time data integration
  • CSV export capability
  • Interactive visualizations

Technology Stack:

  • Backend: Flask + TensorFlow
  • Frontend: Vanilla JavaScript
  • Data: Open-Meteo API + ERA5
